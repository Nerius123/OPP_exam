# Objektinio programavimo (išankstinė) egzamino užduotis

# Programos aprašymas

Ši programa yra skirta analizuoti tekste esančių žodžių dažnumą, poziciją tekste (kurioje eilutėje tas tikras žodis yra) ir nuorodų radimą.
